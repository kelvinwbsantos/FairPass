# TrustCode Template

## Sobre o desafio
Template oficial do desafio TrustCode do Hackathon Web3 RESTIC 29.

## Objetivo
Construir Smart Contracts capazes de automatizar acordos, pagamentos e regras de negócio.

## Exemplos de aplicação
- Escrow
- Pagamentos automáticos
- Royalties
- Governança
- Marketplace descentralizado

## Tecnologias sugeridas
- Solidity
- Hardhat
- Sepolia
- OpenZeppelin
- Ethers.js
- React

## Estrutura
/contracts
/frontend
/scripts
/test
/docs

## Como executar

### Instalar dependências
```bash
npm install
```

### Compilar contratos
```bash
npx hardhat compile
```

### Deploy
```bash
npx hardhat run scripts/deploy.js --network sepolia
```

## Requisitos mínimos
- Contrato deployado
- Fluxo demonstrável
- README funcional
- Vídeo-pitch
